Ce site est un p2 repository Eclipse. Pour installer le plugin :

  1. Help → Install New Software...
  2. Add... puis coller cette URL :
https://bluemind-net.github.io/eclipse-plugin-bm/
  1. Cocher BlueMind Developer Tools → Finish
  2. Redémarrer Eclipse
